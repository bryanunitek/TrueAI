## Version

This manifest is versioned alongside the doctrine folder. The version string is in [`MANIFEST.json`](MANIFEST.json) (`"version": "v1"`). When a new file is added or an existing file is removed or renamed, the version increments and this manifest is updated in the same commit.

## Maintenance

Changes to this folder (new files, renames, deletions) are committed as atomic changes to the TrueAI repository. The MANIFEST.md and MANIFEST.json are updated in the same commit as the files they describe.
